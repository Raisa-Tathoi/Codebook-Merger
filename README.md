# Codebook-Merger
Link: https://raisa-tathoi.github.io/Codebook-Merger/

The aim of this tool is to be able to have 2 codebooks with their hierarchies and code definitions side-by-side 
and merge it into a new codebook that can be then exported as csv

## Specific User Instructions
- Insert child: Drag & Drop onto a code to insert as child under that code
- Merge codes: Hold down shift when dropping onto the code you want to merge with
- Double click on the definition to edit it
